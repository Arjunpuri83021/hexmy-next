## ğŸ”§ Customize Logo:

### Change Colors:

**Navbar.js & Footer.js:**

```javascript
// Play button gradient

// "HEX" color
className="text-cyan-400"
```

### Change Size:

```javascript
// Icon size
className="w-10 h-10"  // Change to w-12 h-12 for bigger

// Text size
className="text-2xl"   // Change to text-3xl for bigger

// Subtext size
className="text-[10px]" // Change to text-[12px] for bigger
```

### Change Glow Effect:

```javascript
// Current glow
className="shadow-lg shadow-cyan-500/50"

// Stronger glow
className="shadow-xl shadow-cyan-500/70"

// No glow
className="shadow-lg"
```
